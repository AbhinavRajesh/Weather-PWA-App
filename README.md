## A Progressive Web App [ PWA ] for Weather Information

### Tech Stack Used
* Visual Studio Code
* JavaScript
* React.JS Framework


### Installation

1. Clone the Repository
    ```shell
    git clone https://github.com/AbhinavRajesh/Weather-PWA-App.git
    ```
    
2. Navigate to Downloaded file
    ```shell
    cd Weather-PWA-App
    ```
3. Install required Dependencies
    ```shell
    npm install
    ```
4. Run the App!
    ```shell 
    npm start
    ```
